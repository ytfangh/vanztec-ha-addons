# Vanztec Home Assistant Add-ons

This repository contains Home Assistant add-ons for secure remote access
and cloud connectivity.

## How to use

1. Open Home Assistant
2. Go to **Settings → Add-ons → Add-on Store**
3. Click the **three dots (⋮) → Repositories**
4. Add this repository URL:https://github.com/ytfang/vanztec-ha-addons
5. Reload the Add-on Store
6. Install the desired add-on

## Available add-ons

### Rathole Client
Secure tunnel client for connecting Home Assistant to Vanztec cloud services.
